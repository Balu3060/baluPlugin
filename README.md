# baluPlugin
MMR Tracker for Rocket League

baluPlugin is an MMR Tracker for Rocket League that saves the streaks locally and lets you track your progress across game sessions with separate tracker for each gamemode.
How to Use:
- click enable plugin
- if you want to save the process after restart, click "Save Progress Locally" 
- select the gamemode you want to play
- if you want to delete the process just press "Reset Stats"
- if the plugin is bugged, just press "Reset Tracker"
